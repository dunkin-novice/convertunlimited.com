JWT ถูกอัปโหลดไหม?

                
ไม่ ถอดรหัสในเบราว์เซอร์

                
ดู exp และ iat ได้ไหม?

                
ได้ timestamp จะถูกแสดงเป็นวันที่ ISO เมื่อมีข้อมูล

                
ควรวาง token production ไหม?

                
ควรหลีกเลี่ยง token production ที่ละเอียดอ่อนหากไม่เข้าใจความเสี่ยง

                
รองรับ Unicode ไหม?

                
รองรับเนื้อหา UTF-8 ใน payload อย่างปลอดภัย
