Supported workflow

URL encoding hữu ích cho query string, API, redirect, tham số tìm kiếm và debug liên kết có khoảng trắng hoặc Unicode.

Khi nào nên mã hóa URL?

URL encoding hữu ích cho query string, API, redirect, tham số tìm kiếm và debug liên kết có khoảng trắng hoặc Unicode.

Dùng URL đầy đủ để giữ cấu trúc URL dễ đọc. Dùng component để mã hóa một giá trị query hoặc đoạn path.

Câu hỏi thường gặp

Công cụ URL này miễn phí không?

Có. Bạn có thể mã hóa, giải mã, sao chép và xóa không cần đăng ký.

URL có được upload không?

Không. Tất cả chạy cục bộ trong trình duyệt.

URL đầy đủ khác component thế nào?

URL đầy đủ giữ : / ? và &, còn component mã hóa cả ký tự dành riêng đó.

Có hỗ trợ Unicode không?

Có. UTF-8, emoji, dấu và chữ không Latin được hỗ trợ.

Percent-encoding lỗi thì sao?

Công cụ hiển thị cảnh báo và giữ nguyên đầu vào.
