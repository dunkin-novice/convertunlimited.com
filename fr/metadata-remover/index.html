Que sont les métadonnées d'image ?

Les métadonnées d'image sont des informations cachées intégrées dans un fichier photo. Ces données sont créées par les appareils photo numériques et les smartphones.

Données EXIF

L'EXIF comprend des détails tels que le modèle de l'appareil, les réglages de l'objectif, la date et l'heure de la prise de vue.

Métadonnées GPS et de localisation

La plupart des smartphones modernes intègrent des coordonnées GPS précises dans les métadonnées de chaque photo que vous prenez.

Comment fonctionne la suppression dans le navigateur

ConvertUnlimited utilise une méthode de ré-encodage via HTML5 Canvas pour garantir que toutes les métadonnées sont éliminées.
