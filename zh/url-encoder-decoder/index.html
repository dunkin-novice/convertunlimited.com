Supported workflow

URL 编码适用于查询字符串、API、重定向、搜索参数，以及包含空格、Unicode 或保留字符的链接调试。

什么时候需要 URL 编码？

URL 编码适用于查询字符串、API、重定向、搜索参数，以及包含空格、Unicode 或保留字符的链接调试。

完整 URL 模式会保留 URL 结构。组件模式适合编码单个查询值或路径片段。

常见问题

这个 URL 工具免费吗？

免费。无需注册即可编码、解码、复制和清空。

URL 会上传吗？

不会。所有处理都在浏览器本地完成。

完整 URL 和组件模式有什么区别？

完整 URL 保留 : / ? 和 & 等分隔符。组件模式也会编码这些保留字符。

支持 Unicode 吗？

支持。UTF-8、emoji、重音和非拉丁文字都支持。

百分号编码错误怎么办？

工具会显示友好警告并保留原始输入。
