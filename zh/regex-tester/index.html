什么时候测试 regex？
Regex 测试可用于表单验证、日志解析、API 文本检查，以及在代码中使用前调试提取规则。
本工具使用浏览器原生 JavaScript RegExp，并把用户文本作为文本渲染，而不是 HTML。

            
常见问题

                
这个 Regex 工具免费吗？

                
免费。无需注册即可测试 pattern 和预览替换。

                
文本会上传吗？

                
不会。Regex 测试在浏览器本地完成。

                
使用哪个 regex 引擎？

                
使用浏览器原生 JavaScript RegExp 引擎。

                
无效 pattern 会怎样？

                
工具会显示友好警告并保留输入。

                
支持 Unicode regex 吗？

                
支持。使用 u flag。
