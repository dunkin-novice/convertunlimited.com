Supported workflow

                
JWT decoder 可在调试 API 和认证流程时查看 token header、payload，以及 exp、iat、nbf 等常见 claims。

            

            
JWT Decoder 能显示什么？
JWT decoder 可在调试 API 和认证流程时查看 token header、payload，以及 exp、iat、nbf 等常见 claims。
解码 JWT 不等于验证 JWT。除非可信系统验证签名和 claims，否则已解码 token 仍可能被伪造或已过期。

            
常见问题

                
会验证 JWT 签名吗？

                
不会。它只解码 header 和 payload。

                
我的 JWT 会上传吗？

                
不会。解码在浏览器本地完成。

                
可以查看 exp 和 iat 吗？

                
可以。存在时会把 Unix timestamp 显示为 ISO 日期。

                
应该粘贴生产 token 吗？

                
除非了解风险，否则避免粘贴敏感生产 token。

                
支持 Unicode claim 吗？

                
支持。UTF-8 payload 会被安全解码。
