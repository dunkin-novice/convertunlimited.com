About ConvertUnlimited

ConvertUnlimited is a browser-native utility platform for common file, image, PDF, and developer workflows. The tools are designed to run in the browser where practical, with trust documentation that explains the difference between the public site and the tracker-free privacy build.

Short answer

ConvertUnlimited is a browser-native utility platform for common file, image, PDF, and developer workflows. The tools are designed to run in the browser where practical, with trust documentation that explains the difference between the public site and the tracker-free privacy build.

What the product is

ConvertUnlimited provides small, practical tools for converting images, compressing files, resizing images, removing metadata, working with PDFs, formatting JSON, encoding data, generating hashes, and similar everyday browser tasks.

The product surface is intentionally simple: open a tool, choose a file or input, process it in the browser when the workflow is supported, and download or copy the result.

Privacy boundary

For supported file-processing workflows, selected file contents are processed locally in your browser. ConvertUnlimited does not provide a server-side upload endpoint for those processing flows.

The public site may load advertising and analytics. Privacy-sensitive workflows should use the privacy build at privacy.convertunlimited.com, which is generated without ads, analytics, remote fonts, or third-party runtime scripts.

How we describe limitations

Browser-native tools depend on browser APIs, available memory, device performance, file format support, and local JavaScript execution. Some large files or unsupported formats may fail locally and require a desktop or server-side workflow.

The Trust Center documents these limits directly instead of relying on broad claims such as guaranteed security or perfect privacy.

Who runs ConvertUnlimited

ConvertUnlimited is an independently built and maintained project. It is not a venture-backed company, an affiliate funnel, or a front-end for someone else's conversion API. One maintainer is responsible for the product decisions, the code, the trust documentation, and the separate privacy build, and that is intentional: the privacy claims on this site only mean something if a single, accountable party stands behind them.

The project began as a reaction to the dominant pattern in free online file tools, where files are uploaded to a server, queued, processed remotely, and sometimes retained. ConvertUnlimited exists to show that a large share of everyday image, PDF, and developer tasks can run entirely in the browser, with nothing leaving the device for supported workflows.

If something on the site does not match what the repository does, that is a bug worth reporting. You can open an issue on GitHub or email info@convertunlimited.com.

How the site is funded

The public site is intended to be supported by advertising so the tools can stay free with no signup and no file limits. Advertising is currently disabled while the site is under review. The privacy build at privacy.convertunlimited.com is, and will remain, free of ads, analytics, remote fonts, and third-party runtime scripts.
