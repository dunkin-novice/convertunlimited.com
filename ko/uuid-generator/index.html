UUID는 언제 사용하나요?

UUID는 테스트 데이터, request ID, 데이터베이스 키, API 예시, 클라이언트 프로토타입에 유용합니다.

이 도구는 browser crypto APIs로 UUID v4를 생성하며 ID를 전송하지 않습니다.
