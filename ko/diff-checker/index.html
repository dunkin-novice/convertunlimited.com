What this tool does

두 버전을 붙여넣고 line-based diff를 실행하며 서버로 보내지 않고 결과를 복사합니다.

Privacy behavior

Selected inputs are processed locally in your browser for this workflow. The public site may load ads and analytics; use the privacy build for privacy-sensitive workflows.

Supported workflow

Diff는 편집 리뷰, snippet 비교, 설정 변경 확인, 텍스트 출력 디버깅에 유용합니다.

Diff Checker는 언제 사용하나요?

Diff는 편집 리뷰, snippet 비교, 설정 변경 확인, 텍스트 출력 디버깅에 유용합니다.

이 도구는 간단한 line-based 비교를 사용하고 텍스트를 HTML이 아닌 텍스트로 안전하게 렌더링합니다.

자주 묻는 질문

이 Diff 도구는 무료인가요?

네. 가입 없이 비교, 복사, 지우기를 할 수 있습니다.

텍스트가 업로드되나요?

아니요. 비교는 브라우저에서 로컬로 실행됩니다.

어떤 diff를 사용하나요?

추가, 삭제, 변경 없음 라인을 표시하는 line-based diff입니다.

공백이나 대소문자를 무시할 수 있나요?

네. 비교 전에 토글을 사용하세요.

Unicode를 지원하나요?

네. Unicode와 이모지를 브라우저 문자열로 처리합니다.

개인정보 처리방침

이 도구에 입력한 텍스트를 수집, 저장, 업로드 또는 전송하지 않습니다.

이용 약관

ConvertUnlimited는 있는 그대로 제공됩니다. 프로덕션 워크플로에서 사용하기 전에 diff 출력을 검토하세요.
