Supported workflow

                
JWT decoder는 API와 인증 흐름을 디버깅할 때 token header, payload, exp, iat, nbf 같은 일반 claims를 확인하는 데 유용합니다.

            

            
JWT Decoder는 무엇을 보여주나요?
JWT decoder는 API와 인증 흐름을 디버깅할 때 token header, payload, exp, iat, nbf 같은 일반 claims를 확인하는 데 유용합니다.
JWT 디코딩은 검증과 다릅니다. 신뢰하는 시스템이 서명과 claims를 검증하지 않으면 디코드된 token도 위조되었거나 만료되었을 수 있습니다.

            
자주 묻는 질문

                
JWT 서명을 검증하나요?

                
아니요. header와 payload만 디코드합니다.

                
JWT가 업로드되나요?

                
아니요. 디코딩은 브라우저에서 로컬로 실행됩니다.

                
exp와 iat를 볼 수 있나요?

                
네. Unix timestamp claims는 ISO 날짜로 표시됩니다.

                
프로덕션 token을 붙여넣어도 되나요?

                
위험을 이해하지 못했다면 민감한 프로덕션 token은 피하세요.

                
Unicode claims를 지원하나요?

                
네. UTF-8 payload를 안전하게 디코드합니다.
