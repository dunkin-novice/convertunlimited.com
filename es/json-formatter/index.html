¿Cuándo formatear JSON?

El JSON formateado es más fácil de inspeccionar, depurar, revisar y compartir. El JSON minificado sirve para payloads de API, ejemplos o configuración.

Esta herramienta valida JSON estricto. No repara JSON inválido ni acepta comentarios, comas finales o claves sin comillas.

Preguntas frecuentes

¿Este formateador JSON es gratis?

Sí. Puedes formatear, validar, minificar, copiar y descargar JSON sin registro.

¿Mi JSON se sube?

No. El análisis y el formato se ejecutan localmente en tu navegador.

¿Soporta Unicode y emojis?

Sí. JSON.parse y JSON.stringify conservan cadenas Unicode y emojis válidos.

¿Puede arreglar JSON inválido?

No. Valida JSON estricto y muestra un error para que lo corrijas manualmente.

¿Qué indentación debería usar?

Dos espacios es común en web, cuatro es más legible y tabs sirve cuando el proyecto lo exige.

Política de privacidad

No recopilamos, almacenamos, subimos ni transmitimos el JSON que pegas. Los archivos abiertos se leen localmente en el navegador.
