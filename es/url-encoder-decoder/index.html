¿Cuándo codificar URLs?

La codificación URL ayuda con query strings, APIs, redirecciones, parámetros de búsqueda y enlaces con espacios, Unicode o caracteres reservados.

Usa URL completa para mantener la estructura legible. Usa componente para un valor de query o segmento de ruta.

Preguntas frecuentes

¿Esta herramienta URL es gratis?

Sí. Puedes codificar, decodificar, copiar y limpiar sin registro.

¿Mi URL se sube?

No. Todo ocurre localmente en tu navegador.

¿Diferencia entre URL completa y componente?

URL completa mantiene : / ? y &, componente también codifica esos caracteres reservados.

¿Soporta Unicode?

Sí. Soporta UTF-8, emojis, acentos y escrituras no latinas.

¿Qué pasa con percent-encoding inválido?

Muestra una advertencia clara y conserva la entrada original.
